Работа с различными данными

Возвращает порядковый номер элемента массива в котором записано указанное значение. (DataInArray)
function DataInArray(Data:Integer; Arr: Array of Integer):Integer; overload;
function DataInArray(Data:Boolean; Arr: Array of Boolean):Integer; overload;
function DataInArray(Data:Double; Arr: Array of Double):Integer; overload;
function DataInArray(Data:String; Arr: Array of String; IgnoreCase:Boolean):Integer; overload;
function DataInArray(Data:String; Arr: TStrings; IgnoreCase:Boolean):Integer; overload;
    Data - искомое значение
    Arr - где искать
    IgonoreCase - если значение Data - строка, то установка этого параметра в true приведет к поиску строк без учета регистра.
    result - порядковый номер элемента, имеющего значение, указанное в Data, либо -1, если такого элемента не найдено.

описано : akDataUtils

Возвращает число, записанное в указанных битах (от..до) (GetBits)
 function GetBits(value:Integer; first, last:Byte):DWORD;
    value - исходное число
    first - номер начального бита (0-31)
    last - номер конечного бита (0-31)
    result - значение, хранящееся в битах first-last числа value.

описано : akDataUtils

Считает CRC32 указанной строки (GetStringCRC)
 function GetStringCRC(Str:String) : Cardinal;
    str - строка по данным которой считается CRC32.    
    result - CRC32 указанной строки.
